Properties¶

Reducibility¶

Accessibility¶

Denoted \(i \to j\) which means

\[\Pr(X_{n}=j\mid X_{0}=i) > 0.\,\]

Communicative¶

Denoted \(i \leftrightarrow j\) and is equivalent with \((i \to j) \wedge (j \to i)\).

Irreducible¶

Denoted \(i \to j, \quad \forall i, j \in S\) which means that it is possible to get to any state from any state

\[\Pr(X_{n}=j\mid X_{0}=i) = P^{n}(i, j) > 0, \quad \exists t \geq 1, \quad \forall i, j \in S\]

Periodicity¶

A state \(i\) has period \(k\) if any return to state \(i\) must occur in multiples of \(k\) time steps. Formally defined

\[k=\gcd\{n>0:\Pr(X_{n}=i\mid X_{0}=i)>0\}\]
  • Aperiodic if \(k = 1\)
  • Periodic if \(k \geq 2\)

Every irreducible and aperiodic Markov chain has a stationary distribution.

Transience and Recurrence¶

Absorbing states¶

A state \(i\) is called absorbing if it is impossible to leave this state.

\[\begin{split}\begin{cases} p_{ii} = 1 \\ p_{ij} = 0 & i \neq j \end{cases}\end{split}\]
